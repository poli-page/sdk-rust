# poli-page

Official [Poli Page](https://poli.page) SDK for Rust — render PDFs from HTML
templates via the Poli Page API.

> **Status**: pre-1.0. This is the Phase 0 scaffold; the public API is being
> ported from the reference Node implementation (`@poli-page/sdk` v1.3). The
> full content of this README lands in Phase 6 alongside the first RC tag.
> See the implementation plan at the repository root.
